홀연히 사라집니다... 입대합니다...

재연은 대한민국 해군 689기 문화홍보부 디자인병으로 지원하여 2023년 1월 30일, 훈련을 위해 진해교육사령부에 입소합니다. 6주 동안의 훈련소 기간에는 연락이 어려워요. 안전하게 다녀올게요. 그동안 재연이 많이 심심할텐데 하고 싶은 말이 있다면 적어주시는게 어떠신가요? 인터넷 편지 작성 방법은 아래에 적어두었어요. 편지 작성이 번거롭다면 게시판에 가볍게 댓글을 남겨주세요.

가볍게 하고 싶은 말이 있다면...

본 게시판 내용은 재연의 친구가 대신 모아서 전달할 예정입니다. 무겁게 하고 싶은 말이 있다면 여기로...

인터넷 편지 작성 방법(2월 8일부터 작성 가능)

  1. 해군교육사령부 인터넷 편지 웹사이트 방문
  2. 약관 동의 후, 편지쓰기 버튼 클릭
  3. 아이핀/휴대폰으로 본인인증
  4. 받는이: 김재연 / 생년월일: 990713 입력 후 편지쓰기 버튼 클릭
  5. 배치 결과에서 김재연 클릭
  6. 편지 작성하기(중도에 날아갈 수 있으니 미리 작성 후 복붙 권장)
  7. 보내기 클릭 후, “등록되었습니다.” 문구 확인하기


유의사항

  • 본인에게만 전달되고 공개되지 않음
  • 하루 1인 1회 1000자 작성 가능
  • 하루 최대 4명까지 작성 가능(인원 초과 시 작성 불가)
  • 편지는 30분 이내에 작성해야 함(시간 초과시 지워짐)
  • 당일 오후 3시까지의 편지가 오후 9시에 전달
  • 재연에게 손편지 답장을 받고 싶다면 본인 주소와 우편번호를 적어주세요.

손편지 발송 방법(2월 8일부터 주소 확인 가능)

  1. 해군교육사령부 홈페이지 접속
  2. 입영장병코너 > 훈련병 페이지
  3. 소대배치 훈련병 / 990713 / 김재연 검색
  4. 나온 주소 빈칸에 붙여놓고 우편봉투 작성
  5. 주소: 경상남도 창원시 진해구 진희로 111 사서함 211-2-3 기초군사교육단 신병교육대대 689기 00대대 00중대 00소대 김재연 (우)51686
  6. 무게에 맞는 우표/라벨 붙이고 발송


유의사항

  • 되도록 규격봉투 + 규격 편지지로
  • 규격 내 종류별 우편 가격
    • 일반우편(2~5일): 400원~450원
    • 등기통상(1~4일): 2500~2550원
    • 일일특급(1일): 3500~3550원
  • 편지 외의 내용물은 절대 발송 불가
  • 우표 부착 유의

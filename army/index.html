인터넷 편지 작성 방법(2월 8일부터 작성 가능)

  1. 해군교육사령부 인터넷 편지 웹사이트 방문
  2. 약관 동의 후, 편지쓰기 버튼 클릭
  3. 아이핀/휴대폰으로 본인인증
  4. 인증 후, 받는이: 김재연 / 생년월일: 990713 입력 후 편지쓰기 버튼 클릭
  5. 배치 결과에서 김재연 클릭
  6. 인터넷 편지쓰기(중도에 날아갈 수 있으니 미리 작성 후 복붙 권장)
  7. 보내기 클릭 후, “등록되었습니다.” 문구 확인하기

유의사항

  • 본인에게만 전달되고 공개되지 않음
  • 하루 1인 1회 1000자 작성 가능
  • 하루에 4명에게만 받을 수 있음
  • 편지는 30분 이내에 작성해야 함(시간 초과시 지워짐)
  • 당일 오후 3시까지의 편지가 오후 9시에 전달
  • 재연에게 손편지 답장을 받고 싶다면 본인 주소와 우편번호를 적어주세요.

손편지 발송 방법(2월 8일부터 주소 확인 가능)

  1. 해군교육사령부 홈페이지 접속
  2. 입영장병코너 > 훈련병 페이지
  3. 소대배치 훈련병 / 990713 / 김재연 검색
  4. 나온 주소 빈칸에 붙여놓고 우편봉투 작성
  5. 주소: 경상남도 창원시 진해구 진희로 111 사서함 211-2-3기초군사교육단신병교육대대 689기 00대대 00중대 00소대 김재연 (우)51686
  6. 무게에 맞는 우표/라벨 붙이고 발송

유의사항

  • 되도록 규격봉투 + 규격 편지지로
  • 규격 내 종류별 우편 가격
    • 일반우편(2~5일): 400원~450원
    • 등기통상(1~4일): 2500~2550원
    • 일일특급(1일): 3500~3550원
  • 편지 외의 내용물은 절대 발송 불가
  • 우표 부착 유의
